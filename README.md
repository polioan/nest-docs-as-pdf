## Commands

```json5
"build": "tsc" // build
"run": "node dist/index" // run
"start": "npm run build && npm run run" // build and run
"format": "prettier --write ." // format
"format:check": "prettier --check ." // check format
```

```shell
# To skip fetching from browser
SKIP_BROWSER=true npm run start
```
